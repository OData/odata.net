# OData Playground

## Microsoft 2023 Hackathon

9/11/2023 - 9/13/2023

## ODataPlayground.Frontend

* Next.js Application

## ODataPlayground.Server

* ASP.NET Application


## Usage

### Only Run the Frontend for Dev

```bash
make frontend-dev
```

Accessible over localhost:3000

### Run the Server and Build Front End

```bash
make
```

Accessible over localhost:5001
